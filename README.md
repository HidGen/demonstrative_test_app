Разные варианты решения одно задачи типа "Лабараторная"
Без проверок и защит от дурака. Без оптимизации производительности

ТЗ:
1. Консольное приложение 
2. 2 команды:
     1) На вход подаётся строка формата: "111 E:\SKL\CRM\NetCore\SKL.Queue\SKL.Queue.Consumer\bin\Debug\net8.0".
В ответ построчным списком выводится список всех файлов в этой папке.
   2) На вход подаётся строка формата: "222 E:\SKL\CRM\NetCore\SKL.Queue\SKL.Queue.Consumer\bin\Debug\net8.0\testoviy_fail.txt".
 В ответ на команду, по указанному пути создаётся файл с именем "testoviy_fail_COMPLETE.txt". Все буквенные символы этого файла изменены на символы верхнего регистра. В консоль достаточно написать сообщение "Файл обработан."
